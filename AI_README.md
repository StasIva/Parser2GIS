## Engineering Process

For every task:

1. Read the engineering documents.
2. Understand the existing architecture.
3. Implement the smallest correct change.
4. Run tests.
5. Perform a self-review.
6. Produce a complete engineering report.
7. Explicitly choose a final task disposition.

---

The engineering documentation is mandatory for all implementation work.
